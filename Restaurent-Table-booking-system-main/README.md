# 🍽️ Restaurant Table Booking System

A full-stack restaurant table booking web application built using Python (Flask) and modern web technologies.

This project was developed using AI-assisted development (Vibe Coding) with tools like Figma, Gemini, and ChatGPT to enhance productivity and problem-solving.

---

## 🚀 Features

- Table reservation system
- Booking confirmation flow
- Database integration (SQLite)
- Clean UI design
- Structured backend using Flask

---

## 🛠️ Tech Stack

Frontend:

- HTML
- CSS
- JavaScript

Backend:

- Python
- Flask

Database:

- SQLite

## ▶️ How to Run Locally

1. Clone the repository
2. Create a virtual environment
3. Install dependencies:
   pip install -r requirements.txt
4. Run the application:
   python app.py
5. Open browser:
   http://127.0.0.1:5000
